[](#getting-started)Getting Started
-----------------------------------

### [](#installation)Installation

1.  Clone the repo
    ```
    git clone https://github.com/edruiz88/react-movie-search
    ```
    
2.  Install NPM packages
    ```
    npm install
    ```
    
3.  Run the app server
    ```
    run: node app
    ```
